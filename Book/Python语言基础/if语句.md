- `and`&`or`

```python
age_0 = 22
age_1 = 18

if (age_0 >= 21) and (age_1 >= 18):
    print(True)

if (age_0 >= 21) or (age_1 >= 21):
    print(True)    
    
print(age_0 >= 23 or age_1 >= 23)
print(age_0 >= 23 and age_1 >= 18)

#True
#True
#False
#False

```

- `in` & `not in`

```python
lettters = ['a', 'b', 'c']

print('a' in lettters)
print('d' in lettters)

print('a' not in lettters)
print('d' not in lettters)

#True
#False
#False
#True
```
